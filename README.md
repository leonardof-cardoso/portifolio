# Portfolio Pessoal

Portfolio pessoal desenvolvido para apresentar minha atuacao como Desenvolvedor de Software, com foco em sustentacao de aplicacoes criticas, troubleshooting avancado, integracoes entre sistemas e projetos reais publicados no GitHub.

O projeto foi construído com HTML, CSS e JavaScript puro, com uma interface moderna, responsiva e organizada para destacar:

- apresentacao profissional
- stack principal
- projetos reais
- acesso direto ao curriculo

## Preview

O portfolio contem:

- secao inicial com resumo profissional
- secao "Sobre Mim" organizada por experiencia, habilidades e objetivo profissional
- trilha animada de tecnologias
- secao de projetos com links para repositorios reais
- secao de CV com download direto do curriculo

## Como Executar

Como este projeto e estatico, basta abrir o arquivo `index.html` no navegador.

Se quiser rodar com um servidor local, uma opcao simples e usar a extensao Live Server no VS Code ou algum servidor HTTP local.

## Personalizacao

Voce pode adaptar facilmente:

- `index.html`: textos, links, secoes e estrutura do portfolio
- `style.css`: cores, layout, animacoes e responsividade
- `script.js`: comportamento do menu mobile
- `curriculo.pdf`: arquivo baixado na secao de CV
- imagens `project*.png` e `image.png`: fotos e previews dos projetos
